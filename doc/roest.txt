ROEST'S NEOVIM                                                    *roest* *roest-nvim*

Welcome back! This is your personal Neovim setup.

If you're reading this after a long break, start here:
    1. Run `:checkhealth` to verify everything works
    2. Read |roest-quickstart| for the essentials
    3. Browse |roest-keymaps| for all your keybindings
    4. Check |roest-plugins| for what each plugin does

QUICK START                                                   *roest-quickstart*

Your most important keymaps (leader is <Space>):

    Finding things~
    <leader>sf      Find files in project
    <leader>sg      Search text (grep) across project
    <leader>/       Fuzzy search in current file
    <leader><leader> Switch between open buffers
    <leader>st      Search all TODOs in project

    Navigation~
    -             Open file browser (Oil)
    <leader>-     File browser in floating window
    <C-h/j/k/l>   Move between splits

    Working with files~
    <leader>a     Add file to Harpoon (your "tabs")
    <C-n>/<C-p>   Next/prev Harpoon file
    <C-e>         Show Harpoon quick menu

    Code intelligence~
    grd           Go to definition
    grr           Find references
    grn           Rename symbol
    <leader>e     Show error popup
    <leader>l     Format code

    Editing~
    sa"           Surround with quotes (visual or motion)
    sd"           Delete surrounding quotes
    sr"'          Replace surrounding " with '

    Completion (insert mode)~
    <Tab>         Next completion item
    <S-Tab>       Previous item
    <CR>          Confirm selection
    <C-Space>     Trigger manually

LEADER KEY                                                      *roest-leader*

Your leader key is <Space>. When you see `<leader>` in docs, press Space.

Example: `<leader>sf` means press Space, then s, then f.
